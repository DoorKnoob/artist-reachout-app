# Artist Reachout Manager

A web application for tracking and managing artist outreach campaigns.

## Features
- Add and manage artist contacts
- Import artists from Excel files
- Customize email templates
- Send personalized emails
- Track outreach status
- Search and filter artists

## How to Use
Simply open the deployed app and start adding artists!

## Deploy to GitHub Pages
This app is deployed using GitHub Pages. Any changes pushed to the main branch will automatically update the live site.
